## 📱 App Overview

Budget Tracker is a personal finance management Android app that helps users track their spending, manage categories, set monthly budgets, and plan for the future. The app is designed to be simple, fast, and private - all data stays on your phone.

**Inspired by research on:** Spendee (simple design), PocketSmith (future planning), and Vault22 (all-in-one dashboard).
